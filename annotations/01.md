## Eslint

- Ao tentar passar parâmetros para um script do package.json, o npm acaba não entendendo, então devo fazer o seguinte

  ```bash
  # Em vez disso
  npm run lint --fix

  # Faço isso
  npm run lint -- --fix
  # Esse -- vai fazer com que esse próximo parâmetro --fix faça parte do script do package.json
  ```
